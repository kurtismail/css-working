Genel kullanım olarak CSS sayfasına marjin, padding 0 verilerek başlanır. Sebebi tarayıcının default değerlerini geçersiz kılmaktır.

Bir Class valuesinin içerisine birden fazla class tanımlanabilir. Style etiketi altında tanımlama yapıldığında bir p etiketine birden fazla class tanımlanmış olur.

Inherit öncelikle bir proporty (özelliğin) inherit olup olmadığına bakılmalı. W3School gibi sitelerden bakılabilir. inherit özelliğe sahip ise bağlı bulunduğu etiketten özeelliği iherit edebilir. örenkte em etiketi içerisine yazılan içerik p etiketinden inherit edilmiş çünkü color ın inherit özelliği bulunmaktadır.
